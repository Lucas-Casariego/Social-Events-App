# *Aplicación de Eventos Sociales*

## Descripción
Este es un proyecto que construí mientras seguía el curso de Udemy "Complete Guide to Building an App with .NET Core and React" de Neil Cummings.

## Características
El objetivo de la aplicación es proporcionar a los usuarios una plataforma social donde puedan crear, administrar y asistir a eventos con otras personas.
La aplicación cuenta con un sistema de autenticación de usuarios que les permite crear cuentas, personalizar sus perfiles y conectarse con otros usuarios.
Los usuarios pueden crear eventos, invitar amigos y administrar los detalles de sus eventos, como la ubicación, hora y fecha. La aplicación también incluye una función de búsqueda que permite a los usuarios encontrar eventos por ubicación, categoría o palabras clave.

## Tecnologías
Para el Frontend, utilicé React y Material UI para crear una interfaz de usuario moderna y receptiva. En lugar de utilizar Semantic UI, opté por usar Material UI por su amplia colección de componentes personalizables que me permitieron crear un diseño consistente en toda la aplicación.

En el Backend, utilicé ASP.NET Core y Entity Framework Core como mi ORM e implementé una API RESTful para manejar las solicitudes del Frontend.

Además, utilicé Vite en lugar de Create React App como herramienta de construcción para mejorar el rendimiento de la aplicación y reducir los tiempos de construcción. Para mejorar las funciones en tiempo real de la aplicación, utilicé SignalR para proporcionar a los usuarios notificaciones e actualizaciones instantáneas sobre eventos e invitaciones.

### Tecnologías utilizadas:
· ASP.NET Core
· Entity Framework Core
· React
· Material UI
· Vite
· SignalR
